== Description ==
Kyros Kebab internal plugin for recipe & ingredient management.
- Custom post types: Ingredient, Recipe
- Live calculator (shortcode [recipe_calculator id="123"] and block)
- Roles: krm_admin, krm_operation, krm_supervisor
- Settings: currency, unit system, allergen master list, misc %
- Print-friendly SOP export (browser print)
- Malay translation ready

== Installation ==
1. Copy folder to wp-content/plugins/
2. Activate.
3. Assign roles to users.
4. Configure Settings (Recipe Ops â†’ Settings).
5. Add Ingredients, then Recipes.
6. Embed calculator via shortcode or block.
